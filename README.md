# 十日终焉 (Doomsday)

一个关于十日终焉的同人Minecraft模组。

## 默认按键绑定

- `V` - 使用回响
- `G` - 切换连续回响模式
- `]` - 下一个回响
- `[` - 上一个回响

## 配置文件

模组包含两个主要配置文件：

### doomsday.toml (通用配置)

- 回响装配
  - `healthReduction`: 装配眼球时减少的最大生命值 (默认: 6)
  - `sanityReduction`: 装配眼球时减少的最大理智值 (默认: 200)
- 本地聊天
  - `enable`: 是否启用本地聊天功能
  - `range`: 本地聊天的可见范围（方块）(默认: 32.0)

### doomsday-echoes.toml (回响配置)

包含多种回响的详细配置：

#### 无垢回响
- 基础施法距离：32格
- 冷却时间：12000 ticks (10分钟)
- 理智消耗：150

#### 爆燃回响
- 冷却时间：6000 ticks (5分钟)
- 理智消耗：50
- 可通过提升信念等级获得强化效果

所有参数都可以在配置文件中自定义调整。
